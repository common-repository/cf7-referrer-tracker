Automatically include visitor\'s tracking information (from http headers) in Contact Form 7 Emails. 

== Description ==
This plugin is an extension to the popular \"Contact Form 7\" plugin, it saves value of HTTP REFERER and other http headers to cookies on first user visit and adds these info to mail body when contact form 7 is submitted.
This plugin allows website owners to get more information about the visitors who are submitting contact forms on their website. Several tracking information (like HTTP Referer) can be tracked and automatically included in contact form submission emails.

<strong>Works out of the box. Just install and activate the plugin, thats it!</strong>

== Installation ==
1. Upload the \'Referer-Tracker-For-Contact-Form-7\' folder to the \'/wp-content/plugins/\' directory via ftp or any other means.
2. Activate the plugin through the \'Plugins\' menu in WordPress.
3. (Optional) Go to \"Settings->Referer Tracker For Contact Form 7\' menu to set which headers to track. That is all.


== Screenshots ==
1. Select which HTTP Hearders to track and Save.
2. Selected headers will be automatically included in all contact form submission emails.
